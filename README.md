# HackerTerminal

Консольный текстовый квест на C# (.NET 10), стилизованный под терминал хакера.

## Сюжет

Студию PMR Peak Games наняли тебя, чтобы найти того, кто слил в сеть
предрелизную сборку игры «Порог» за неделю до релиза.

## Запуск

```
dotnet run
```

или открыть `HackerTerminal.csproj` в Rider и запустить.

## Команды

`help`, `ls`/`dir`, `cd <папка>`, `cat <файл>`/`open`, `decrypt <файл> <ключ>`,
`unlock <код>`, `hack <цель>`, `scan`, `connect <адрес>`, `status`, `save`,
`clear`/`cls`, `exit`/`quit`.

## Прохождение (шпаргалка)

1. `cat readme.txt`, затем `cd inbox`, `cat notes.txt` — узнать сдвиг шифра проходной (6).
2. `cd ..`, `cd reception`, `decrypt badge.enc 6` — получить код `COMPASS`.
3. `unlock compass` — уровень 2, открыта сеть студии.
4. `scan` — найти адрес студии, `connect 172.20.14.9`.
5. `cd studio`, `cat security_log.txt`, `cat team_chat.json` — в переписке
   команды разработчик случайно присылает пароль от билд-сервера: `midnight`.
6. `hack admin midnight` — уровень 3, открыт архив студии.
7. `scan`, `connect 10.55.2.1`, `cd archive`, `cat keycard.txt` — ключ 3.
8. `decrypt truth.enc 3` — финал, личность утечки установлена.

Часть файлов на каждом уровне — просто отвлекающие: список покупок,
журнал посетителей, бэклог спринта, инструкция к кофемашине. Их можно
прочитать, но подсказок в них нет. 

## Структура проекта

- `Program.cs` — точка входа.
- `Game.cs` — главный цикл, разбор команд, вся логика уровней.
- `GameState.cs` — состояние игрока (уровень, очки, подключения).
- `QuestFile.cs` — модель файла игровой файловой системы.
- `Cipher.cs` — шифр Цезаря.
- `save.json` — создаётся командой `save`, в git не коммитится.
